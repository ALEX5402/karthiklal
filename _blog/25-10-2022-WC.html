About this project

● This is a simple set of script to clean all of your windows trash files and to repair some windows corrupted files.
● It is intended to be used as a tool to clean up your windows environment.
● It is not intended to be a replacement for a full-functional tool.
● Every commands on this script can be executed by just typing the command name. (i made it easily that's it.)
● This script is not a malware or virus. It is just a simple script to clean your windows environment.
● This is a free script. You can use it for free.

How to use this script ?

Method 1

● Download the latest release of this project from [GITHUB].
● Run the clean-windows.exe file [Forced to work only on Administrative Privilege Mode].
● Wait for the setup to complete [It will take some time].

Method 2

● git clone https://github.com/karthik558/clean-windows.git or Download the repo from GitHub [ZIP]
● Open the clean-windows folder.
● Run the clean-win.bat file [Forced to work only on Administrative Privilege Mode].
● Wait for the setup to complete [It will take some time].
